Using Ractive with Browserify

Home » Tips » Using Ractive with Browserify

Browserify is a way of using node-style requires in the browser, bundling your scripts into a single file for efficient deployment.

You can use Ractive.js alongside Browserify using Ractify by marcello3d (thanks!). Visit the Ractify repository for downloads and documentation.
